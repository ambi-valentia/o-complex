# A mobile friendly marketplce web app written in React+Typescript

Contents:
- Products-reviews page
- Cart page

Features: 
- Mobile friendly
- Products from Amazon's API
- Cart and phone number get memorized and appear after reload
- Phone number field is masked
- Field validation
